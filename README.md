## サイト概要
## サイトテーマ
キャンプ準備や共有ができるコミュニティーサイト

## テーマを選んだ理由
キャンプに行きたい人、行き始めたばかりだけど何を準備したらいいかわからない人がいると思います。
その人たちが手軽にいけるようなコンテンツがあればいいと思い着手しました。

また行ったキャンプの投稿や、他の人とキャンプを共有し、次にいくときの参考になればと思います。

## ターゲットユーザ
キャンプ初心者や興味がある人

##主な利用シーン
キャンプに行く前の準備、行った後の共有

設計書
